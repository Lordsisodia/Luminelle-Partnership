# Per-Page Token Map (reference only)

Landing
- Hero: bg.heroGradient, text.primary, text.muted, accent.cta, accent.ctaText, accent.badge, border.subtle.
- Value/FAQ/Leaderboard: bg.surface or bg.subtle, border.subtle, text.primary/muted.
- CTA strips: accent.cta/ctaText, promo.highlight for badges.

Product (PDP)
- Hero/media: bg.surface, border.subtle, text.primary; price/add-to-cart: accent.cta/ctaText; badges: accent.badge.
- Reviews/accordions: text.primary/muted, border.subtle, bg.subtle.
- Promo callouts: promo.highlight.

Blog
- Index cards: bg.surface, border.subtle, text.primary/muted; tags: accent.badge.
- Post body: text.primary; links: accent.cta.

Account
- Headers/sections: text.primary, bg.surface, border.subtle; buttons: accent.cta/ctaText; alerts (if any): state.*.

Admin
- Tables/cards: bg.surface, border.subtle, text.primary/muted; filters/pills: accent.badge; primary actions: accent.cta/ctaText; status chips: state.success/warning/danger.

Out of scope
- Shopify Checkout (external); keep native styling.
