# Recommended Color Taxonomy

Target: 14 semantic color types cover our surfaces/components without overfitting.

1) text.primary — main copy
2) text.muted — secondary text
3) text.inverse — on-dark text
4) bg.canvas — page background
5) bg.surface — cards/blocks
6) bg.subtle — lightly tinted areas
7) bg.heroGradient — hero/CTA gradients
8) border.subtle — dividers/borders
9) accent.cta — primary buttons/fills
10) accent.ctaText — text on CTA
11) accent.badge — pills/chips
12) promo.highlight — special highlights/testimonials
13) state.success — positive states
14) state.warning/state.danger — caution/error states

Optional extras (add if needed):
- focus.ring — focus outline color
- shadow.soft — shadow token referencing brand hue
- bg.overlay — modal overlays
