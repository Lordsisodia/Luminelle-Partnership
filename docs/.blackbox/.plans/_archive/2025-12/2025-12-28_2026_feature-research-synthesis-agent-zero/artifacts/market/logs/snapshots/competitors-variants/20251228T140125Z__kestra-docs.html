What is Kestra?

Kestra is an open-source, infinitely-scalable orchestration platform that enables all engineers to manage business-critical workflows declaratively in code.

Thanks to 1000+ plugins and embedded Code editor with Git and Terraform integrations, Kestra makes scheduled and event-driven workflows easy.

Whats possible with Kestra

Orchestrate Data Pipelines

Schedule, backfill, automate, and scale data pipelines declaratively

Orchestrate dbt Workflows

Version-control, test, and deploy dbt models with GitOps and on-demand compute

Coordinate Microservices

Run microservice tasks in response to events, recover from failures, and scale as needed

Automate Infrastructure

Provision resources, manage builds, and scale infrastructure workflows

Automate Manual Approval Processes

Integrate Human-in-the-Loop approvals into critical steps of automated workflows

Orchestrate Python Workflows

Automate, schedule and scale Python workflows in a declarative way
