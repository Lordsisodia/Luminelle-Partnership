# Artifacts (Run Outputs)

This folder holds the “hard outputs” of a single run:
- raw model output
- sources/citations
- extracted structured data
- short synthesis
- run metadata (reproducibility)

Recommended:
- `run-meta.yaml`
- `sources.md`
- `summary.md`
- `extracted.json` (optional)
- `raw.md` (optional; may be gitignored)
