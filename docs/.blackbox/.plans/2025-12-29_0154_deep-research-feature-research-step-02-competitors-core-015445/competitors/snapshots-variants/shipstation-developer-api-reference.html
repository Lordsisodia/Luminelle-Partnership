ShipStation API V1

ShipStation API V1 is available for certain plans and allows developers to build applications that interface with the ShipStation platform. The V1 API endpoints can be used to automate many tasks including:

  • Managing Orders
  • Managing Shipments
  • Creating Shipping Labels
  • Retrieving Shipping Rates
