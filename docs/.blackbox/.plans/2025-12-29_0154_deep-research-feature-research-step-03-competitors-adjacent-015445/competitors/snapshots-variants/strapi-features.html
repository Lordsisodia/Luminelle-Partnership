Internationalization (i18n)

Manage and publish content in multiple locales, allowing for localized content strategies and broader audience engagement.

Blocks Editor

Drag and drop rich text in a user-friendly WYSIWYG environment, making content creation seamless and straightforward.

Content Management

Dynamic Zones

Editors adjust layouts with dynamic components for flexible page design.

Content Management

Content History

No more headaches over lost edits or accidental changes. Keep your workflow smooth and stress-free with Content History.

Live Preview

Live preview changes before publishing to enable collaboration and confident content validation.

Customization

Conditional Fields

Give editors a smarter, more intuitive contentediting experience by showing only the fields that matter.

Create APIs

Content GraphQL API

The GraphQL API allows performing queries and mutations to interact with the content-types through Strapi's GraphQL plugin.

Security

API Tokens

Manage end-user access to your API securely with customizable tokens, ensuring that data exposure is controlled and intentional.

Security

TypeScript Support

Improve coding practices and reduce errors with full TypeScript support, ensuring a more secure and maintainable codebase.

Security

Audit Logs

Keep a detailed record of every action taken within your CMS, allowing for better security monitoring and compliance.

Security

RBAC

Control access and permissions within your CMS with advanced role-based access control, enhancing security and compliance.

Single Sign-On (SSO)

Simplify user authentication with Single Sign-On from major third-party providers, streamlining access while maintaining security.
