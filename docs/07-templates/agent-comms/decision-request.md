# ❓ Decision Request (fast, non-annoying)

Use this when the agent is blocked or a fork matters.

## 2) Why it matters (one sentence)

This changes **<what>** because **<reason>**.

## 3) Options (2–4)

1) Option A — <pros/cons in one line>
2) Option B — <pros/cons in one line>
3) Option C (optional) — <pros/cons in one line>

## 4) Recommendation

Recommend: **Option <A/B/C>** because **<1–2 reasons>**.

## 5) Evidence

- `<path/to/artifact.md>`
- `<path/to/triage-table.md>`

## 6) What happens next

If approved, I will: **<next 1–3 actions>**.
