## Files

- Prefer `kebab-case.md` for most docs.
- Use dates for time-based docs: `YYYY-MM-DD-topic.md` or `topic-YYYY-MM-DD.md`.

## Folders

- Avoid spaces in new folder names.
- If a folder would exceed ~10 children, add a grouping folder (`runbooks/`, `assets/`, `notes/`) instead.
